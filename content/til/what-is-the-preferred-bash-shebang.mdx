You should use **`#!/usr/bin/env bash`** for [portability](<https://en.wikipedia.org/wiki/Shebang_(Unix)#Portability>): Different \*nixes put `bash` in different places, and using `/usr/bin/env` is a workaround to run the first `bash` found on the `PATH`.
