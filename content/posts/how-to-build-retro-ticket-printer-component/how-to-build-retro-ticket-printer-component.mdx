## Rolling Receipt

The `RollingReceipt` component animates the receipt rolling out from the slot, with perforated edges and absolute positioning. It uses custom keyframes and gradients for a convincing paper effect.

### Visual Demo

<RollingReceipt show={true}>Animated receipt content</RollingReceipt>

**Obstacle:**
Simulating a paper roll-out using only CSS and React.

**Trick:**
Use absolute positioning, custom keyframes, and CSS gradients for perforation. The receipt is clipped and animated to appear as if it's rolling out of the slot.

## Ticket Base

The `TicketBase` component provides the main visual container for the ticket printer card. It handles layout, border, background, and overall structure, ensuring the card is perfectly square and visually balanced.

### Visual Demo

<TicketBase>Content goes here</TicketBase>

**Obstacle:**
Getting the aspect ratio and border radius to feel retro but modern.

**Trick:**
Use Tailwind’s `aspect-square` and `rounded-2xl` for a perfect square and soft corners. A subtle gradient adds depth.

## LCD Header

The `LCDHeader` component displays a retro-styled label at the top of the card, flanked by decorative screws.

### Visual Demo

<LCDHeader />

**Obstacle:**
Achieving a retro LCD look with minimal markup.

**Trick:**

## Decorative Screws

To complete the retro look, decorative screws are added to the card corners using simple styled divs.

### Visual Demo

<DecorativeScrews />

**Obstacle:**
Making basic divs look like metallic screws.

**Trick:**
Use `rounded-full`, subtle gradients, and inner shadows to create a 3D metallic illusion.

## Print Button

The `PrintButton` component is designed to feel tactile and responsive, mimicking arcade-style hardware buttons.

### Visual Demo

<PrintButton />

**Obstacle:**
Making the button feel physical and interactive.

**Trick:**
Combine Tailwind gradients, shadows, and an `active:scale` effect to simulate a button press. Bold typography and icons reinforce the retro vibe.
