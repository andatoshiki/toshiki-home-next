# toshiki-home-next

> Toshiki's next-generation personal all-in-one portfolio, blog, homepage, and internet hideout built with Next.js 14, React 18, and TypeScript. It combines MDX/Velite-powered writing for the blog and TIL sections, project and repository showcases, guestbook comments, music and coding statistics dashboards, RSS feeds, and interactive experiences like a 3D room, journey globe map, masonry and album galleries, and media library tracking for anime, manga, games, and books.
